# 🎉 Fest Management System

## Overview
A Python-based Fest Management System developed as part of my CBSE Class XII curriculum. The project was inspired by my experience helping organize my school's Annual Fest as the Head of the Cultural Department, where I realized how useful a simple system could be for managing participants and event details.

## Features
- Participant Registration
- Event Management
- Record Management
- Menu-driven console interface

## Technologies Used
- Python

## Concepts Used
- Functions
- Loops
- Conditional Statements
- File Handling

## Learning Outcomes
This project helped me strengthen my Python programming skills while showing how technology can be used to solve real-world organizational challenges.

## Future Improvements
